# NLP4J
These LAPPSGrid services implement components from [EmoryNLP's NLP4J](https://emorynlp.github.io/nlp4j/). The components use pre-made configuration files to run the NLP4J commands, while the customizable services allow users to specify configuration as parameters.

## Available Services
- [Custom Train](md/customtrain.md)
- [Custom Decode](md/customdecode.md)
- [Tokenization](md/tokenizer.md)
- [Part-of-speech Tagging](md/tagger.md)
- [Named Entity Recognition](md/ner.md)
- [Dependency Parsing](md/dependency.md)
